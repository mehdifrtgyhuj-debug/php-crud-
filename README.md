# 🚀 User Management System (CRUD)

A simple and efficient **User Management System** built with PHP that allows full CRUD operations (Create, Read, Update, Delete).

---

## 📌 About the Project

This project demonstrates a basic backend system for managing users. It is designed to practice core PHP development and database operations.

---

## ✨ Features

- ➕ Add new users  
- 📋 View users list  
- ✏️ Edit user information  
- ❌ Delete users  
- ⚡ Simple and fast structure  

---

## 🛠 Tech Stack

- PHP  
- HTML  
- CSS  
- MySQL (if used)

## 🎯 Purpose

This project was created to practice:

- Form handling in PHP
- Basic backend logic
- Database interaction
