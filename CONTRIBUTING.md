# Contributing Guide

## سپاسگزاریم برای علاقه‌مندی به مشارکت! 🙏

شما می‌توانید به بهبود این پروژه کمک کنید. اینجا راهنمایی برای مشارکت است:

### 📋 قوانین مشارکت

1. **Fork کنید** - مخزن را فورک کنید
2. **Branch بسازید** - شاخه جدید برای ویژگی خود بسازید
3. **Commit کنید** - تغییرات خود را Commit کنید
4. **Push کنید** - به شاخه خود Push کنید
5. **PR بسازید** - Pull Request باز کنید

### 🎯 انواع مشارکت

- **Bug Fixes** - باگ‌ها را ثابت کنید
- **Features** - ویژگی‌های جدید اضافه کنید
- **Documentation** - مستندات را بهبود دهید
- **Tests** - تست‌های جدید بنویسید

### 📝 Commit Messages

```
فرمت: [نوع] توضیح کوتاه

انواع:
- feat: ویژگی جدید
- fix: رفع باگ
- docs: بهبود مستندات
- style: تغییر سبک کد
- refactor: بازسازی کد
- test: اضافه کردن تست
```

### ✅ قبل از ارسال PR

- [ ] کد را تست کردید
- [ ] مستندات را بروز کردید
- [ ] سبک کد را رعایت کردید
- [ ] تنها تغییرات مرتبط را اضافه کردید

### 🤝 رفتار احترام‌آمیز

- احترام به نظرات دیگران
- بحث‌های سازنده
- کمک هم‌دیگر برای یادگیری

متشکریم! 💪
